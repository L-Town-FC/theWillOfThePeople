!ceelo start [bet]: Creates a joinable game of Ceelo with for [bet] gbp. You only have 15 seconds to enter the game

!ceelo join: Lets you join a game

!ceelo roll: Lets you roll the die

!ceelo wash: Lets you roll the die, but the hand doesn't count

!ceelo status [game]: Lets you see the status of the specified game. If no game is specified, it shows a list of all actve games

!ceelo basics: Gives a list of basic rules and hand values
